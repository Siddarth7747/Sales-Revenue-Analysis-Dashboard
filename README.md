# 📊 Sales & Revenue Analysis Dashboard

## 📌 Project Overview
This project is a Power BI dashboard designed to analyze sales and revenue data using the SuperStore dataset. The dashboard provides interactive visualizations, KPI tracking, and business insights to help understand sales performance and profitability.

## 🎯 Objectives
- Analyze sales and revenue trends
- Track key business KPIs
- Identify top-selling products
- Compare category-wise and region-wise performance
- Enable interactive filtering using slicers

## 📈 Dashboard Features

### 💰 Key Performance Indicators (KPIs)
- Total Sales
- Total Profit
- Total Orders

### 📊 Visualizations
- Revenue Trend Analysis
- Sales by Category
- Sales by Region
- Top Selling Products

### 🎛️ Interactive Filters
- Region Slicer
- Category Slicer

## 🛠️ Tools Used
- Microsoft Power BI Desktop
- SuperStore Sales Dataset
- Data Visualization Techniques

## 📂 Dataset Information
The dashboard uses a retail sales dataset containing:
- Order Details
- Product Information
- Sales and Profit Data
- Customer Segments
- Regional Information

## 💡 Business Insights
- Identify high-performing products and categories
- Monitor revenue growth over time
- Compare regional sales performance
- Support data-driven business decisions
